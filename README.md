# マイクラ建築Mod解説サイト

このリポジトリは、マインクラフトの建築系Modの導入方法や便利な使い方を解説するウェブページのソースコードです。
霧雨の美しい景観とともに、あなたの理想の建築をサポートする情報をまとめています。

##  開発・技術構成

個人でコツコツとHTML/CSSを組み立てて制作しています。

*   **HTML5 / CSS3**: ライブラリを使わず、セマンティックなマークアップとカスタムCSS（背景デザインやフレックスボックスなど）で構築。
*   **ディレクトリ構成**:
    *   `index.html` / `index.css`: エントランス・トップページ
    *   `home.html` / `home.css`: メインのホーム・ナビゲーションページ
    *   `images/`: サイト内で使用するマイクラのスクリーンショット等の画像フォルダ
